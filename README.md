# Cartola Bancária Parser

## Introduction
This project is designed to parse and process financial data from Cartola Bancária, facilitating users in analyzing their transactions and balances.

## Features
- **Data Parsing**: Efficiently extracts transaction data from Cartola Bancária formatted documents.
- **Data Analysis**: Allows users to perform statistical analysis on their financial data, including income, expenses, and balance tracking.
- **User-Friendly Interface**: Easy-to-navigate interface for loading files and viewing parsed data.

## Installation
To install the project, please clone the repository and run the setup script.

```bash
git clone https://github.com/EduardoFerrerC/cartola-bancaria-parser.git
cd cartola-bancaria-parser
python setup.py install
```

## Usage
To use the parser, run the main script with your Cartola Bancária file as input:

```bash
python main.py /path/to/cartola_file.txt
```
